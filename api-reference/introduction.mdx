Добро пожаловать в документацию по API Квант. Здесь вы найдете информацию обо всех методах, сможете ознакомиться с инструкциями по их использованию и узнать, какие данные они возвращают.

API Квант разработан с учетом архитектурного стиля REST, предлагает логично структурированные URL, предназначенные для работы с ресурсами, принимает запросы с данными в формате `application/json`, отвечает данными в формате JSON, а также следует стандартам использования HTTP статус-кодов, механизмов аутентификации и HTTP методов.

## Использование API

REST API Квант разработан так, чтобы его было просто использовать. Чтобы лучше ознакомиться с API, вы также можете использовать инструменты вроде [Postman](https://www.postman.com/) или [Insomnia](https://insomnia.rest/).
