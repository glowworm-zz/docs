title: 'Введение в API Квант'
---
API Квант представляет собой RESTful API, который позволяет вам взаимодействовать с документами Квант:

<ul>
  <li>Просмотр и поиск документов Квант</li>
  <li>Создание новых документов и копирование существующих</li>
  <li>Общий доступ и публикация документов</li>
  <li>Исследование страниц, таблиц, формул и элементов управления</li>
  <li>Чтение, вставка, добавление, обновление и удаление строк</li>
</ul>
---
## Использование API

REST API Квант разработан так, чтобы его было просто использовать. Вы можете выбрать любой язык и платформу для отправки запросов. Чтобы лучше ознакомиться с API, вы также можете использовать инструменты вроде [Postman](https://www.postman.com/) или [Insomnia](https://mintlify.com/docs/api-playground/mdx/configuration).

## Аутентификация

API Квант использует API-ключи для проверки и авторизации запросов. Вы можете управлять и просматривать свои API-ключи через панель управления Квант, или запросить ключ через поддержку. Вам нужно передать свой API-ключ в заголовке авторизации с префиксом Bearer при отправке запроса.

```json
Bearer ВАШ_API_КЛЮЧ
```
