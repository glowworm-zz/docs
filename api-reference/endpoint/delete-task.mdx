---
title: 'Удаление коммуникации'
openapi: 'DELETE /api/tasks/{id}'
---

Метод `DELETE /api/tasks/{id}` позволяет удалить существующую задачу в системе Квант. Необходимо передать в строке запроса идентификатор задачи, которую вы хотите удалить.

Удаление задачи является необратимой операцией, поэтому используйте этот метод с осторожностью.

### Параметры запроса

Для получения информации о задаче, необходимо указать идентификатор задачи в пути запроса:

- `id` (number, required) - уникальный идентификатор задачи.

### Пример запроса
```json
DELETE /api/tasks/12345
```

### Успешный ответ
При успешном удалении задачи сервер вернет HTTP статус 204, не возвращая тело ответа.

```json
HTTP/1.1 204 No Content
```

### Ошибка
Если запрос не может быть обработан из-за неверных данных, возвращается ошибка с соответствующим описанием.

- 400 (Bad Request):
```json
{
  "error": "Ошибка в запросе",
  "message": "Неверные параметры запроса"
}
```

- 401 (Unauthorized):
```json
{
  "error": "Неавторизован",
  "message": "Необходима аутентификация"
}
```
- 403 (Forbidden):
```json
{
  "error": "Доступ запрещен",
  "message": "У вас нет доступа к этой задаче"
}
```
- 404 (Not Found):
```json
{
  "error": "Задача не найдена",
  "message": "Задача с указанным ID не существует"
}
```
- Ошибка 429 (Too Many Requests):
```json
{
  "error": "Слишком много запросов",
  "message": "Превышен лимит запросов"
}
```
